## Anatomy

Color anatomy is to ensure consistency and harmony across design elements

### Platform Colors

Platform colors make up majority of the user interfaces within the Discovery Education platform. Neutral makes up our Typography colorization and several other areas of our platform. Interface should be considered for most user interface colorization, while Interactive should be considered and utilized for all elements that can be interacted with, such as links and buttons.

### Shade Layering

When utilizing colors in user interfaces you should tend to stack coloring in layers from dark to light or light to dark. This creates depth when layering colors on top of each other and allows for other specific needs such as page depth and interactions.

![Shade Layering Visual](@assets/shade-layering.png)

### Contrast and Accessibility

Nebula aims to meet WCAG stands for color contrast for all essential content and design elements.

| Score | Contrast Ratio | Result                            |
| ----- | -------------- | --------------------------------- |
| AAA   | 7.0+:1         | Passes WCAG 2.1 AAA               |
| AA    | 4.5:1          | Passes WCAG 2.1 AA                |
| AA18  | 3.0:1          | Passes WCAG 2.1 AA for Large Text |
| DNP   | < 3.0:1        | Does Not Pass                     |
